# NLDC_Analysis-and-Control-of-a-Genetic-Toggle-Switch-

This repository contains the project developed for the **Nonlinear Dynamics and Control** course.  
The work focuses on the modeling, analysis, and control of a **genetic toggle switch**, a bistable system in synthetic biology, with particular attention to nonlinear behavior and resource competition effects.

## Overview
The project investigates the dynamic behavior of the toggle switch through equilibrium analysis, stability assessment, and bifurcation-related considerations. Different control strategies were implemented and compared in order to study the regulation of the system under different operating conditions.

## Methods
The developed analysis includes:
- mathematical modeling of the toggle switch dynamics
- equilibrium and local stability analysis
- nonlinear control design and performance evaluation

The implemented control approaches include:
- PI control
- input-output feedback linearization
- sliding mode control
- reinforcement learning based methods

## Objective
The main goal of the project is to analyze the bistable dynamics of the genetic toggle switch and evaluate different control techniques in terms of regulation capability, robustness, and overall system behavior. A key control objective is to stabilize the system around the unstable equilibrium point, so as to regulate the toggle switch near a desired operating condition.
